1. **Запуск тестов**:
   - Перейдите в раздел **Actions** на GitHub.
   - Выберите workflow **UI Tests**.
   - Нажмите **Run Workflow** для запуска тестов.

2. **Просмотр отчета**:
   - Перейдите в раздел **Actions** на GitHub.
   - Выберите workflow **pages build and deployment**.
   - Дождитесь завершения **Deploy** и перейдите по ссылке
